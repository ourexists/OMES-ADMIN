介绍
-----------------------------------
一套管理工业设备的平台生态。包括了OMES-admin管理平台、OMES智设备（移动端）。  
平台的主要目的是形成一套工业层面的设备生命周期管理、设备数据采集及分析、设备巡检、设备健康预测、场景化管理的适用于工程化的项目
