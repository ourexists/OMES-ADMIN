# OMES 工业化设备管理系统

## 一、项目概述

**OMES** 是一套面向工业制造场景的**边缘侧 MES（制造执行系统）** 与管理平台，用于车间设备连接、数据采集、产线/工单/物料/计划管理及多租户运维。支持 OPC UA、Modbus、MQTT、REST、S7、WINCC 等多种工业协议，实现“边缘采集 + 统一门户”的管理方式。

> 更多说明见：<https://blog.ourexists.site/2026/01/30/omes/>

## 二、业务功能

### 2.1 车间与设备管理

- **车间管理**：按车间、产线组织生产区域，维护车间层级与归属关系。
- **设备台账**：管理设备基本信息、设备组，支持设备运行、在线、告警等状态记录与快照。
- **设备采集配置**：配置车间级、设备级数据采集方式，与 SCADA/PLC 等外部系统对接，获取实时运行数据。

### 2.2 连接与数据采集

- **多协议连接**：支持 OPC UA、Modbus TCP/RTU、MQTT、REST、S7、WINCC、BACnet 等工业协议，统一配置连接地址、认证与采集周期。
- **采集绑定**：将设备与采集通道绑定，按设定的周期或表达式定时采集数据。
- **独立采集节点**：支持部署独立采集服务，通过 MQTT 等协议接收/上报数据，实现“边缘采集、集中管理”。

### 2.3 生产计划与工单

- **主生产计划**：编制主生产计划，关联产线/工位，管理计划明细、手册。
- **工单下发**：根据计划生成制造工单及明细，下发给产线或设备执行。
- **计划—工单—产线—设备闭环**：从计划、工单、产线到设备执行，形成计划下发、执行反馈与数据回写的闭环。

### 2.4 物料与 BOM

- **物料主数据**：维护物料主数据及分类。
- **BOM 管理**：维护物料清单（BOM）及层级结构，支撑工单用料与计划排产。

### 2.5 产线与工位

- **产线管理**：维护产线及工位拓扑，为计划、工单提供产线维度。
- **工位关联**：工单、主计划可按产线/工位进行关联与下派。

### 2.6 定时任务与配方

- **定时任务**：按 cron 表达式配置周期任务，用于数据采集、同步或其它业务调度。
- **配方管理**：管理生产配方及任务类型，支撑工艺参数下发与追溯。

### 2.7 多租户与权限

- **租户隔离**：支持多租户，租户间数据隔离，适合集团多厂区、SaaS 等场景。
- **组织架构**：维护部门、平台、账号、角色，支持部门与用户关联。
- **权限控制**：基于角色、部门、平台的细粒度权限，控制页面、接口及数据访问。

### 2.8 消息与通知

- **站内消息**：提供消息与通知，支持运维提醒、告警推送。
- **设备告警**：设备异常、告警可与消息联动，便于运维与追溯。

### 2.9 SCADA 与看板

- **SCADA 配置**：支持车间级 SCADA 服务配置（服务地址、认证、映射、采集间隔），与第三方 SCADA 或看板系统对接。
- **实时数据**：提供车间、设备实时采集数据接口，支撑大屏、看板与监控展示。

## 三、应用场景

| 场景             | 说明 |
|------------------|------|
| **离散/流程制造** | 在车间边缘部署 OMES，统一管理产线、工单、计划、物料与设备，形成边缘 MES 能力。 |
| **多协议采集**    | 对接 PLC、SCADA、传感器等，通过 OPC UA、Modbus、MQTT 等协议采集设备数据。 |
| **集团多厂区**    | 多租户隔离，支持多工厂、多车间在同一套系统中分别使用与运维。 |
| **看板与大屏**    | 与 SCADA、看板系统集成，提供实时采集数据接口，用于车间大屏、监控看板。 |

## 四、产品形态

- **管理端**：提供 Web 管理界面，覆盖设备、产线、工单、物料、连接、采集、SCADA、消息、用户等完整管理功能，是“一站式”边缘管理入口。
- **采集/同步端**：可独立部署的数据采集或同步服务，通过 MQTT 等协议接收或上报数据，适合现场分布式采集场景。
