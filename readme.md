介绍
-----------------------------------
本平台聚焦工业设备管理领域，构建涵盖设备全生命周期管理、统一数据采集与治理、实时分析与建模的技术体系；基于智能巡检与设备健康评估模型，实现预测性维护与故障预警；并通过场景化建模与业务编排能力，支撑复杂工业场景下的数字化运营与决策优化


> 项目介绍：https://blog.ourexists.site/2026/01/30/omes/


平台生态:
-----------------------------------
 * OMES-admin管理平台
 * OMES智设备（移动端）
 * OMES-VIDEO (边缘图像检测)。
